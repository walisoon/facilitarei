# Sistema Facilita

Sistema de gestão de crédito desenvolvido para facilitar o processo de análise e aprovação de crédito.

## Tecnologias

- Next.js 14
- TypeScript
- Supabase
- Tailwind CSS

## Deploy

O sistema está hospedado no Netlify e usa o Supabase como backend.
